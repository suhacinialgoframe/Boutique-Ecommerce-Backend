# Boutique E-Commerce Backend

A REST API backend for a boutique e-commerce application built using FastAPI and MongoDB.

## Tech Stack

- Python
- FastAPI
- MongoDB
- PyMongo
- Docker
- Git & GitHub

## Features

### Product Management
- Create products
- View products
- Search products
- Filter products by price

### Category Management
- Create and manage categories

### Customer Management
- Customer registration and management

### Cart Management
- Add products to cart
- Remove products from cart
- View cart items

### Order Management
- Create orders
- Track order status

### Payment Management
- Create payments
- Update payment status

### Advanced Features
- Pagination
- Sorting
- Searching
- Aggregation pipelines
- Dashboard statistics
